# ⏱️ 高精度ミリ秒タイマーシステム 操作マニュアル

本システムは、作業時間をミリ秒（1/1000秒）単位で正確に計測し、自動的に担当シートへ記録するための社内専用ツールです。

---

## 1. クイックスタートガイド（基本の流れ）

基本操作は **「開く」「測る」「記録する」** の3ステップだけで完結します。

```
[ブックを開く] ──> 自動でシート選択＆タイマー起動
                     │
                     ▼
[計測スタート] ──> 作業実施（セル入力等も自由に可能）
                     │
                     ▼
[記録して終了] ──> 自動でB1セルに保存＆タイマー終了

```

### 📥 ステップ 1：ブックを開く

1. SharePointから対象のExcelブックを「デスクトップアプリで開く」で起動します。
2. 許可されたメンバーであれば、ご自身の名前が入ったシートが自動的に選択（アクティブ化）され、画面上にタイマー画面が自動で立ち上がります。

* ※すでに「B1セル」に記録がある場合は、上書き防止のため自動起動はスキップされます。

### ⏱️ ステップ 2：計測（スタート／ストップ）

* **【スタート】**：`スタート` ボタンをクリックすると計測が始まります。
* **【一時停止】**：計測中に `ストップ` ボタンをクリックすると、タイマーが一時停止（時間は維持）します。
* **キーボード操作**：ボタンにフォーカスがある状態なら、**[Space] キーまたは [Enter] キー**を押すだけで「スタート／ストップ」の切り替えが可能です。

### 💾 ステップ 3：記録して終了

1. 計測を終えたいシートが開いている（画面に見えている）ことを確認します。
2. タイマー上の **`記録して終了`** ボタンをクリックします。
3. 確認ダイアログで **[はい]** を選択すると、今開いているシートの「B1セル」に計測時間が保存され、タイマーが安全に終了します。

* 記録が完了すると、Excel左下のステータスバーに「〇〇シートに記録しました」と完了通知が表示されます。

---

## 2. 操作画面（ユーザーフォーム）の見方

タイマー画面の各ボタンと表示の意味は以下の通りです。

| コントロール名 | 説明 |
| --- | --- |
| **時間表示パネル** | 現在の経過時間を `時:分:秒.ミリ秒` でリアルタイム表示します。<br>

<br>※**計測時間が「10分」を超えると、数字が自動的に「赤色」に変化**し警告します。 |
| **スタート／ストップ** | タイマーの始動と一時停止を行います。交互にキャプションが切り替わります。 |
| **リセット** | 計測を一時停止し、時間を `00:00:00.000` にクリアします。 |
| **記録して終了** | 現在の計測時間を表示中のシートのB1セルに保存し、画面を閉じます。 |
| **操作マニュアルを開く** | クリックすると、**タイマーを安全に自動一時停止**した上で、ブラウザを起動して本マニュアルを表示します。 |

> 💡 **キーボード達人向けのヒント（Tabキー移動）**
> マウスを使わずに **[Tab]** キーを押すと、フォーカスが `[スタート/ストップ] ➔ [リセット] ➔ [記録して終了]` の順でループ移動します。キーボードだけで完璧な時間管理が可能です。

## 3. 知っておくと安心な仕様（エラーではありません）

### Q. セルに文字を入力している間、タイマーが止まって見えるのですが？

* **A. 正しい動作（Excelの仕様）です。ご安心ください。**
Excelは「セルに文字を入力している間（編集モード）」、画面の更新を一時的に止める仕様になっています。しかし、**裏ではミリ秒単位の時計が1秒の狂いもなく動き続けています。** 入力を終えて `Enter` キーを押した瞬間に、正しい時間へ一瞬で同期（追いつき）します。

### Q. 記録確認などの「ダイアログ」を開いている間、時間はズレますか？

* **A. ズレません（自動時間補正機能つき）。**
「記録して終了」や「×（閉じる）」ボタンを押して「本当に終了しますか？」などの確認画面（ダイアログ）が出ている間、Excelのプログラムは一時的に停止します。
しかし本システムは、**「確認ダイアログが表示されていた時間」をミリ秒単位で自動計算し、キャンセルして戻った際にスタート時間を過去にさかのぼらせて再開する補正機能**が組み込まれています。

## 4. トラブルシューティング（困ったときは）

### 🚨 状況：ブックを開いたが、タイマーが自動で起動しない

以下の3つのケースが考えられます。順に確認してください。

1. **B1セルにすでに時間が記録されている場合**
* **原因**: 既存データを誤って消去・上書きしないための安全機能です。
* **対策**: 再度計測し直す場合は、一度B1セルの値を削除（空欄に）してから、以下の「手動起動」を行ってください。


2. **あなたのユーザー名がシステム（許可リスト）に登録されていない場合**
* **原因**: 新メンバー追加やPCのアカウント名変更が原因です。この場合、Excel左下のステータスバーに **「【タイマー】ユーザー名「〇〇」は許可リストに未登録です…」** と警告が表示されます。
* **対策**: 管理者へ名前の追加を依頼してください。それまでの間は、以下の「手動起動」を行うことで通常通り計測と記録を行うことができます（作業が止まることはありません）。


3. **マクロ（VBA）がブロックされている場合**
* **原因**: セキュリティ設定によりマクロの実行が止められています。
* **対策**: Excel上部に「セキュリティ警告：マクロが無効にされました」と表示されている場合は、**「コンテンツの有効化」** または **「編集を有効にする」** をクリックしてください。



---

## 🛠️ タイマーを手動で起動する方法

自動起動しなかった場合でも、以下の手順でいつでもタイマーを起動できます。

1. キーボードの **`[Alt] + [F8]`** キーを同時に押します。
2. マクロ一覧ダイアログが表示されるので、**`ShowTimerForm`** を選択します。
3. **`[実行]`** ボタンをクリックします。

* ※手動起動した場合、「今画面に開いているシート」に対して時間が記録されます。

---

**管理・メンテナンス担当者へのお願い：**
メンバーの追加・削除は、VBAエディタ（`Alt + F11`）を開き、`ThisWorkbook` の最上部にある `ALLOWED_USER_LIST`（カンマ区切り文字列）を書き換えるだけで安全に行えます。
